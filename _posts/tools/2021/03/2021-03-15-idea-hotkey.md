# IntelliJ IDEA

## 기본 키맵

### 일반

|                                                      |             Mac OS            |            ⊞ Windows            |
|:-----------------------------------------------------|:-------------------------------|:--------------------------------|
| 해당 도구 창 열기                                     | ⌘ 0 ... ⌘ 9                    | Alt + #[0-9]                    |
| 모두 저장                                             | ⌘ S                            | Ctrl + S                        |
| 동기화                                                | ⌘ ⌥ Y                          | Ctrl + Alt + Y                  |
| 에디터 최대화 토글                                     | ⌘ ⇧ F12                        | Ctrl + Shift + F12              |
| 현재 프로필로 현재 파일 검사                           | ⌥ ⇧ I                          | Alt + Shift + I                 |
| 현재 체계의 빠른 전환                                  | ^ §, ^ `                       | Ctrl + BackQuote (`)            |
| 개방형 설정 대화상자                                   | ⌘ ,                            | Ctrl + Alt + S                  |
| 프로젝트 구조 대화상자 열기                            | ⌘ ;                            | Ctrl + Alt + Shift + S          |
| 동작 찾기                                             | ⌘ ⇧ A                          | Ctrl + Shift + A                |
